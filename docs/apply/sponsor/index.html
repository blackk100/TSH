Flexible Sponsorships

We realise that not all companies are the same, and hence allow a flexible sponsorship approach.

  • Central
  • Local

Sponsor for all events on all the days a given programme, while the programme is active.

 

Cover the following expenses:

  • Media Coverage
  • Logistics
  • Badges/Pins/T-shirts
  • Banners/Posters/Flags
  • Event-specific expenses (shared with Local Sponsors)

Sponsor for one or more events on one or more days a given programme, while the programme is active.

 

Cover the following expenses:

  • Refreshments for volunteers and other participants
  • Flyers/Pamphlets
  • Event-specific expenses (shared with Central Sponsors)
